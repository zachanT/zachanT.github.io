Projects

Song Quiz 2.0

Simple game that plays a snippet of a random song and has the player guess the name and artist. Points are awarded for correctly guessing either the name or artist. One game consists of five rounds where each player takes a turn guessing.

Enjoyed playing a similar song quiz game with my family but we always wished there was more customization with the music categories that we could select. I utilized the Spotify API to allow players to use their own playlists as 'categories'.

Express + React + Node + Spotify API

Frideshare

Application to keep track of items in a user's fridge or pantry with the ability to share user's fridge and shopping list. A user can invite members to join their group by entering their email. Once a user is in a group you can see other members' shopping list as well as items in the fridge that they are willing to share.

Initial frontend was done as part of a design class with a small team of two other students. Added additional features and implemented a backend on my own after the class ended.

PostgreSQL + Express + Handlebars + Node

Productoro

Productoro is a new Pomodoro Timer application. Using the Pomodoro technique, Productoro breaks down work in to intervals called pomos as a means to enhances user time management and increase user productivity and focus.

Worked with classmates to follow the Agile development process as a software development team. This included planning, designing, developing, and testing.

Vanilla HTML + CSS + Javascript

Social Media Site
